```
dots - Dotfile management made less toilesome.

USAGE:
    dots [OPTIONS] <SUBCOMMAND>

FLAGS:
    -h, --help    Prints help information

OPTIONS:
        --store <PATH>    Path of dotfile store [default: ~/.dotfiles]

SUBCOMMANDS:
    add       Link dotfile to target in home directory
    help      Prints this message or the help of the given subcommand(s)
    list      List all installed dotfiles in the given store
    remove    Unlink dotfile from target in home directory
```
